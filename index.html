Strava Dashboard

Configura tu app una vez y comparte el link con quien quieras

1
Crea tu app en Strava

Ve a strava.com/settings/api → crea una app.
En Authorization Callback Domain escribe el dominio de tu GitHub Pages:
tuusuario.github.io

2
Edita el archivo index.html

Abre el archivo, busca la línea // ← CAMBIA ESTO
y reemplaza 237317 con tu número de Client ID.

3
Sube a GitHub Pages

Crea un repo en GitHub → sube el archivo → activa GitHub Pages.
Tu link será: tuusuario.github.io/nombre-repo

4
¡Listo!

Entra al link, haz clic en "Conectar con Strava" y autoriza la app.
Funciona en cualquier ordenador, sin instalar nada.

El token se guarda solo en tu navegador (localStorage).
Nadie más puede ver tus datos.
